Q. How to install webpack server?

Ans- Using "npm install webpack-dev-server --save-dev" command and putting ("start": "webpack-dev-server --mode development --open") inside Scripts in package.json file as shown below:
"scripts": { "dev": "webpack --mode development", "build": "webpack --mode production", "start": "webpack-dev-server --mode development --open" },
and devServer: { contentBase: './dist' }, in webpack.config.js file. and to start sever use this command - "npm run start".

Q. How to install HTML webpack Plugin?

Ans- Using "npm install html-webpack-plugin --save-dev" command and putting ("start": "webpack-dev-server --mode development --open") inside Scripts in webpack.config.js file as shown below:
plugins: [ new HtmlWebpackPlugin({ filename: 'index.html', template: './src/index.html' }) ],
